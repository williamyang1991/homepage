Image Stylization

TET-GAN: Text Effects Transfer via Stylization and Destylization

AAAI 2019
Shuai Yang, Jiaying Liu, Wenjing Wang and Zongming Guo.

This work presents a novel Texture Effects Transfer GAN (TET-GAN), which are trained to disentangle content and style features of text effects images to accomplish both the objective of style transfer and style removal. To support the training of our network, we propose a new text effects dataset with as much as 64 professionally designed styles on 837 characters.

Logo Detection System

Digital Image Processing, 2016
Shuai Yang, Yanghao Li and Yicheng Huang

Implemented SIFT-based and deep-based image detection algorithm.

System Functions

  • View dataset by category
  • View detection results and the groundtruth
  • Single image detection and batch detection
  • Caculate mean average precision
  • Three different algorithms to select

Algorithms

  • SIFT-based: SIFT features are matched between the test image and training images to estimate affine transformations. The masks of training images are warped onto the test image and then accumulated to form a confidence map. The logo is extracted using the region growing algorithm over the confidence map.
  • Multi-Task Deep Networks: Images are divided into K*K grids. Deep networks based on Inception-BN is used to classify each grid.
  • Fast R-CNN: Implemented the paper: Ross Girshick. Fast R-CNN. ICCV 2015.

Simple Video Processing System

Digital Video Processing and Analysis, 2014
Shuai Yang, Li Meng and Chong Ruan

Implemented a video processing system that provides vide coding/decoding, stabilization and denoising functions.

Video Codec

  • Use ffmpeg to decode every frame from the video
  • Convert the AVFrame to IplImage, then stabilize or denoise the image
  • Convert the handled IplImage to AVFrame then write to a raw video
  • Encode the raw video

Video Stabilization

  • A pyramidal version of Lucas-Kanade optical flow computation
  • A six-parameters affine transformation model
  • A Gaussian motion smoothing

Denoising

  • Non-Local Means filter

e-Course

Computer Networks (Honor Track), 2014
Shuai Yang, Yuan Huang, Gaoxiang Zhang and Ran Wang

CPU Simulator

Computer Architectures, 2014
Shuai Yang, Yuan Huang and Gaoxiang Zhang

We learned MIPS and how CPU ran machine codes in pipeline this semester. We were asked to use C++ to simulate a CPU.

Our program reads in assembler codes(which will then translated into machine codes) or machine codes and simulates how a CPU runs them. We provide CPUs with and without pipeline. In pipeline mode, our program handles data hazard and control hazard. The MFC GUI makes our program easy for users to understand the computer operational principle.

Taiko

Java Programming, 2013
Shuai Yang, Yiming Wu, Yiqing Xue and Zheng Huang

Our team members like to play the music game OSU!. Therefore we developed our own Taiko.

Game User Interface

The drumbeats come out from both left side and right side, with characters (A, S, ;, ') on them. Players shall push corresponding buttons when drumbeats moving into the centre circle.

Evaluation System

We evaluate players' performances according to valid drum beats, the precision and combos. We have five ranks: S, A, B, C, D.
